Navigation Layout Solutions

Current Issue: Navigation links appear left-aligned instead of centered on desktop.

Solution 1: CSS Grid (Recommended)

LOGO
Artwork
Education
Collaborate
About
Contact
Spacer
.navbar-container {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    align-items: center;
    gap: 2rem;
}

.navbar-logo {
    justify-self: start;
}

.navbar-links {
    justify-content: center;
    grid-column: 2;
}

.navbar-mobile-toggle {
    display: none; /* Hidden spacer on desktop */
    grid-column: 3;
}

Solution 2: Flexbox with Equal Spacing

LOGO
Artwork
Education
Collaborate
About
Contact
Hidden
.navbar-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.navbar-logo,
.navbar-mobile-toggle {
    flex: 0 0 200px; /* Equal width */
}

.navbar-links {
    flex: 1;
    display: flex;
    justify-content: center;
}

@media (min-width: 768px) {
    .navbar-mobile-toggle {
        visibility: hidden; /* Keep space but hide */
    }
}

What the Fix Does:

  • Creates a balanced 3-column layout on desktop
  • Centers navigation links in the middle column
  • Keeps logo on the left
  • Maintains responsive behavior for mobile
